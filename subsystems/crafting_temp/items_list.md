## Complex Provenance Examples (Legendary Tier)

These 7 examples showcase the deepest provenance tracking capabilities of the system. Each requires multiple layers of specific crafting history.

### 1. Wolf Hart Man of Steel (Legendary Sword)
**Natural Language Recipe:**
"A sword whose blade is steel made from iron ore mined with a manasteel pickaxe, whose handle is oak wood, and whose pommel is a wolf heart from a wolf slain with a weapon whose blade is manasteel."

**Provenance Chain:**
```
Wolf Hart Man of Steel (sword instance)
├─ blade: steel_bar
│  └─ made from: iron_ore
│     └─ mined with: pickaxe (with manasteel head)
├─ handle: oak_planks
└─ pommel: wolf_heart
   └─ from: wolf (carcass)
      └─ slain with: weapon (any)
         └─ whose blade is: manasteel_bar
```

**Crafting Steps:**
1. Mine iron_ore using a manasteel pickaxe
2. Smelt steel_bar from the special iron_ore + coal at forge
3. Craft/obtain a weapon with manasteel blade
4. Slay wolf using the manasteel weapon
5. Harvest wolf_heart from wolf carcass
6. Craft sword with: steel blade, oak handle, wolf_heart pommel

### 2. Moonblade of the Shadow Hunt (Legendary Sword)

```
Moonblade (sword instance)
├─ blade: moonsilver_bar
│  └─ crafted at: moonlight_shrine
├─ handle: elder_logs (any)
└─ pommel: shadow_essence
   └─ from: shadow_elk (carcass)
```

## Complexity Analysis

These 7 legendary items demonstrate:

1. **Depth**: 3-5 levels of provenance recursion
2. **Tool Requirements**: Specific tools with specific materials used in gathering
3. **World Object Requirements**: Specific crafting stations or kill locations
4. **Temporal Requirements**: Time-of-day or lunar cycle checks
5. **Multi-Source Requirements**: Same creature providing multiple components
6. **Ritual Requirements**: Specific sequences (build on altar location, feed before slaying)
7. **Material Transformation**: Infusion, blessing, cursing as crafting steps

### Quality Progression Path

```
Crude Quality (Starter):
  Materials: stone, basic wood, basic hide
  Tools: stone pickaxe, stone hatchet, stone knife
  Weapons: stone dagger, wooden spear
  Armor: basic leather cap
  
Common Quality (Early):
  Materials: copper, tin, bronze, iron, oak, leather
  Tools: bronze/iron pickaxe, iron hatchet, iron knife
  Weapons: bronze/iron sword, iron spear, oak bow
  Armor: leather body, iron coif
  
Uncommon Quality (Mid):
  Materials: steel, willow, deer_hide
  Tools: steel pickaxe, steel smithing_hammer
  Weapons: steel longsword, steel battleaxe, willow bow
  Armor: steel chestplate, steel coif
  
Rare Quality (Advanced):
  Materials: mithril, yew, bear_pelt
  Tools: mithril pickaxe, mithril chisel, ritual_dagger
  Weapons: mithril longsword, yew bow, crossbow
  Armor: mithril chestplate, mithril plate_helm
  
Epic Quality (Endgame):
  Materials: manasteel, moonsilver, phase_silk, elder_wood
  Tools: manasteel pickaxe, blessing_wand
  Weapons: manasteel sword, crossbow with steel limbs
  Armor: plate_body with mithril plates, phase_silk padding
  
Legendary Quality (Ultimate):
  Materials: blessed/cursed variants, ritual materials
  Requires: Complex provenance chains
  Examples: Wolf Hart Man of Steel, Moonblade, Cursed Reaper
  All 7 complex provenance examples from previous section
```

### Critical Path Analysis

**To craft anything:**
1. Need basic tools (crude pickaxe, hatchet, knife) ← stone + wood

**To progress beyond crude:**
2. Need mining → copper + tin → bronze OR iron
3. Need forge + anvil → smelt ores → bars
4. Need bronze/iron tools → better gathering

**To reach endgame:**
5. Need mithril + special materials → manasteel/moonsilver
6. Need special crafting stations → moonlight_shrine, blessed_shrine, sacrificial_altar
7. Need ritual tools → ritual_dagger, blessing_wand
8. Need specific provenance chains → legendary items

**Bootstrap Sequence:**
```
1. Hand-gather stone + wood (no tools needed)
2. Craft crude pickaxe + hatchet
3. Mine copper + tin + iron
4. Gather more wood
5. Build forge + anvil
6. Smelt bronze/iron bars
7. Craft common-tier metal tools
8. Continue progression...
```
